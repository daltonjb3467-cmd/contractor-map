GoHighLevel Integration

Sync contractors as contacts & opportunities. Requires a Cloudflare Worker proxy.

Setup Steps:
1. Get your GHL API key from Settings → Business Profile
2. Get your Location ID from the URL bar in GHL
3. Deploy the Cloudflare Worker (see ghl-worker.js file)
4. Paste the Worker URL above
5. Click "Test Connection", then "Load Pipelines"
